📋 NETLIFY'YE DEPLOYMENT İÇİN BAŞLAMAK:

1. FORMSPREE ID'İ GÜNCELLEYIN (5 dakika)
   ▶ https://formspree.io adresine gidin
   ▶ Hesap oluşturun
   ▶ Yeni form oluşturun
   ▶ Form ID'nizi kopyalayın
   ▶ index.html satır 1301'de "REPLACE_WITH_YOUR_FORMSPREE_ID" 
     kısmını ID'niz ile değiştirin

2. GITHUB'A PUSH EDIN (2 dakika)
   ▶ Terminal'de:
     cd c:\xampp\htdocs\aysenur-portfolio
     git add .
     git commit -m "Update Formspree ID"
     git push origin main

3. NETLIFY'DE DEPLOY EDIN (2 dakika)
   ▶ https://app.netlify.com açın
   ▶ GitHub ile giriş yapın
   ▶ "New site from Git" tıklayın
   ▶ aysenur-portfolio deposunu seçin
   ▶ "Deploy site" tıklayın
   ▶ BITTI! 🎉


🚀 DEPLOYMENT SONRASI:

• Netlify size bir URL verecek: https://[random-name].netlify.app
• Bu URL'de siteniz canlı olacak
• Her GitHub push'unda otomatik deploy olacak
• Form gönderişleri Formspree'ye gelecek

💡 İPUÇLARI:

• Portföy içeriğini güncellemek istersen:
  1. index.html'i düzenle
  2. Kaydet
  3. git push origin main
  4. Netlify otomatik deploy eder (1-2 dakika)

• Özel domain eklemek istersen:
  1. Netlify Dashboard → Domain settings
  2. "Add custom domain" tıkla
  3. DNS ayarlarını takip et

• Form gönderişlerini görmek için:
  1. Formspree dashboard'unu açıp
  2. "Submissions" sekmesine bak


🆘 SORUN GİDERME:

Form çalışmıyorsa:
  ▶ Formspree ID'si doğru mu kontrol et
  ▶ Browser console'ı aç (F12)
  ▶ Spam klasörü kontrol et

Site 404 gösteriyorsa:
  ▶ Browser cache'i temizle (Ctrl+Shift+Del)
  ▶ Netlify deploy loglarına bak
  ▶ Publish directory "/" olmalı

Görseller yüklenmiyorsa:
  ▶ Görsel yolları göreceli olmalı (images/profile.jpg)
  ▶ Görseller .gitignore'da olmamalı
  ▶ Tüm görseller git'de commit edilmiş olmalı
